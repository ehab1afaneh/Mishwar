ASN.1 (Abstract Syntax Notation 1) is the language introduced by the CCITT (now ITU-T) on Recommendations X208 and X209, and chosen by the ISO to describe the datatypes used in SNMP.
Using ASN.1 a standard author can define simple primitive datatypes (like integer), complex constructed ones and macros . Recommendation X209 defined a set of Basic Encoding Rules (BER) to translate data values into a string of octets for transmission across a network.

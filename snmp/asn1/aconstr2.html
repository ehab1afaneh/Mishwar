Constructors allow to define complex structures from primitive types. The constructors used in SNMP are :

    SEQUENCE : An ordered list of datatypes.
    SEQUENCE OF : An ordered list of the same type (example : sequence of integers)

Datatype definitions in ASN.1

The ASN.1 datatype definitions are written in the following form :
     DatatypeName ::= DEFINITION

By convention, each word in the datatype name starts with a capital letter.
example :
RequestID ::= INTEGER

Constructor datatypes definition

This definition shows that the Message datatype is made up of three fields : an INTEGER, an OCTET STRING and a third type called PDU.
example :
Message ::=
	SEQUENCE {
		version		INTEGER { version -1(0) },
		community	OCTET STRING,
		pdu		PDU,
	}

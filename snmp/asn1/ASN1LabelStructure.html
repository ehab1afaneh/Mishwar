Encoding Examples : 
IPAdress ::=
	[APPLICATION 0] -- in network-byte order
	IMPLICIT OCTET STRING (SIZE(4))
    [APPLICATION 0] shows that this definition have an identifier APPLICATION of reference 0.
    So the identifier of this type is 0100 0000 (40 in Hex).
    • Class Application 01
    • Primitive Form 0
    • Reference 0-0000
Example : GetNextRequest-PDU 
GetNextRequest-PDU ::= [1]
	IMPLICIT SEQUENCE {
		requestId RequestID,
		errorStatus ErrorStatus, -- always 0
		errorIndex ErrorIndex,   -- always 0
		variable-bindings VarBindList
		}

    The identifier given is [1], no more precision, so it is a CONTEXT-SPECIFIC.
    The identifier is then 1010-0001 (A1 in Hex.)
    • Class Context Specific 10
    • Constructed Form 1
    • Reference 0-0001

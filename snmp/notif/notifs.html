System informations and preferences

  • Browser version :
    The tutorial contains some Java applets.
    It's better to use :
    Communicator ( Netscape 4.0 ) by Netscape
    Explorer 4.0   by Microsoft

  • Screen :
        • You should use a 800x600 size (conventional window size) with 256 colors minimum to maximize the capabilities of the tutorial.
        • Configure your browser for the default Netscape and Explorer font size 12 . It is necessary to keep in good terms with your eyes.

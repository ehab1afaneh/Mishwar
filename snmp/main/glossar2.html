Gauge - A datatype that measures a quantity that increases and decreases, such as a queue length.

Gateway - An IP router. Many RFC documents use the term gateway rather than router .

Get-bulk-request (version 2) - A request for a mixture of individual and repeated (usually tabular) values.

Get-request - A message that requests the value of one or more MIB variables.

Get-next-request - Enables a manager to retrieve values sequentially. One popular use of the get-next-request is to read through the rows of a table.

Get-response - Version 1 response to a get-request, get-next-request, or set-request.

Group - A named set of closely related MIB definitions within a module.


Inform-request - Manager-to-Manager message containing a notification.

Interior Gateway Protocol (IGP) - used to exchange routing information within an autonomous system.

Interface - Software and hardware that enables a protocol's Network Layer to transmit Protocol Data Units onto a transmission medium. An interface may be made up of several sublayers.

International Organisation for Standardisation (ISO) - An international body founded to promote international trade and cooperative progress in science and technology.

International Telecommunication Union- Telecommunications Standardization Sector (ITU-T) - Organisation that creates recommendations for international voice and data communications protocols and regulations.

International Telegraph and Telephone Consultative Committee (CCITT) - Original name for the ITU-TS. An organisation formed to coordinate the connection of telephony and data communications facilities into international networks.

internet - A network made up of subnetworks connected by one or more routers.

Internet - A worldwide collection of interconnected computer networks, it is based on the TCP/IP protocol suite.

Internet Architecture Board (IAB) - Board that oversees the Internet protocol development and standardisation process.

Internet Assigned Numbers Authority (IANA) - The authority responsible for controlling the assignment of a variety of parameters, such as well-known ports, multicast addresses, terminal identifiers, and system identifiers.

Internet Control Message Protocol (ICMP) - Protocol used to report problems in delivering IP datagrams. ICMP also provides several useful query services.

Internet Engineering Steering Group (IESG) - A group that coordinates the activities of the IETF working groups.

Internet Engineering Task Force (IETF) - A group directed by the IAB, charged with solving immediate Internet problems.

Internet Protocol (IP) - The TCP/IP layer 3 protocol responsible for transporting datagrams across an internet.

Internet Research Task Force (IRTF) - A group directed by the IAB, charged with long-term research on Internet protocols.

IP Datagram - The unit of data routed by IP.


Local Area Network (LAN) - A network that covers a small geographical area. Typical LAN technologies include Ethernet and Token-Ring.


Macro - A facility for defining templates that expand the ASN.1 language.

Management Interface Base (MIB) - A logical database made up of the configuration, status and statistical information stored at a device.

Manager - Person who is in charge of a Network, person who enter orders in the monitor. And it can also be a software in a network management station that enables the station to send requests to view or update MIB variables, and to receive traps from an agent. In version 2, also supports sending and receiving a a device.

Media Access Control (MAC) - A protocol governing a station's access to a network. For example, CSMA/CD provides a set of MAC rules for sending and receive Data across a local area network.

Metropolitan Area Network (MAN) - A technology supporting high-speed networking across a metropolitan area. IEEE 802.6 define a MAN protocol.

MIB II - A set of managed object definitions aimed at managing TCP/IP-based internets.

MIB Group - A collection of closely related objects.

MIB Subtree - The set of nodes that are children of a node in the tree of OBJECT IDENTIFIER.

Module - A named collection of MIB datatype definitions, e.g, for managing a specific technology such as Ethernet, a specific protocol family such as Net Ware products protocols, or a specific application such as electronic mail.

Monitor - A device that listens to all traffic on a LAN or on a wide area kink, gathering statistics, and capturing traffic that matches some specific criteria.


Node - Endpoint of a network connection or a junction common to two or more lines in a network.


OBJECT IDENTIFIER - a string of numbers derived from a global naming tree, used to identify an object.

OSI - Operating System Interconnect (task force)

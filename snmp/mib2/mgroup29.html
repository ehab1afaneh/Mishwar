The Transmission Group(1.3.6.1.2.1.10)

In fact, the Transmission group is not a group. It is just a node position in the MIB-II tree. Many MIBs, each containing groups that apply to different transmission technologies, are attached below this node.






dot3 : IEEE 802.3 (Ethernet)
dot4 : IEEE 802.4 (factory broadband)
dot5 : IEEE 802.5 (Token Ring)

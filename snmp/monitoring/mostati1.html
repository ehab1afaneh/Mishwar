(1.3.6.1.2.1.16.1)

This group defines global statistics measured for a whole LAN (or WAN ).
An entry in the table of an interface reports detailed statistics for this interface connected to another. You can start with a few report statistics in the table and more entries for multiple interfaces.

Some terms in the MIBs are different from what is used in the sniffer (software of the management station showing the statistics). MIBs use Packet and FCS (Frame Check Sequence) where sniffer use Frame and CRC (Cyclic Redundancy Code) .
In those statistics, we have traffic counts such as Total Frames and Total Bytes. These parameters are counted from the time the monitor was started. And the MIB statistics table starts at 0 from the moment that a valid entry is created.

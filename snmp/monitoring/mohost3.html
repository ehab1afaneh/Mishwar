(1.3.6.1.2.1.16.4)

With the address stored in the MAC Frame, a monitor of a LAN can detect all the active host of the LAN. It can also get an idea of how much traffic (incoming and outgoing) each host is generating.

The host group consist of the following parts :

  • The host control table : it is used to configure which of its attached LAN / WAN the monitor have to watch. If all the tables are filled, the monitor starts to delete entries. If the status of this control table is set to invalid, the table is deleted.

  • The Host Table : this table gets filled when the operator sets to a valid entry in the Host Control Table. Hosts are add to the table as soon as they are discovered their traffic is tracked.

  • The Host Time Table : it contains the same information as the Host Table. The difference is that instead of being indexed by physical address, it is indexed by the hostTimeCreationOrder.

(1.3.6.1.2.1.16.8)

As its name says, this group contains the parameters that control the capture of packets that match with a preset filter. Buffer Control Table configure the buffer memory used for the packet capture.

Operators can decide how much memory to allocate for capture and even to warp over or stop capture when the memory is exhausted. It is not compulsory to store the entire packet, a table variable can be configure to choose the number of bytes to save.

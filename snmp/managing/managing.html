A lot of interfaces can be managed by SNMP, it would be too long here to deal with all of them.
The ones that will be treated are :

 

 

 

Here is a non exaustive list of interfaces that can be managed by SNMP :

  • Ethernet Interface
  • Token Ring Interface
  • FDDI Interface
  • RS-232-Like Hardware Interface
  • PPP Interface
  • DS1/ES1 (T1 or E1) Interface
  • DS3/ES3 (T3 or E3) Interface
  • X.25 Interface
  • Frame Relay and SMDS Interfaces
